## AWS CLI のインストール

ubuntu 24 以降では apt でインストールできないので注意

公式手順では snap のみのサポートになっている（もしくは curl）

> Because AWS doesn't maintain third-party repositories other than snap, we can’t guarantee that they contain the latest version of the AWS CLI.
>
> <https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html#getting-started-install-instructions>

パッケージマネージャーで入れづらいので注意だな


# 11/10

## zizmor

zizmor

https://github.com/zizmorcore/zizmor

GitHub Actions のワークフローの静的解析ツールの一つ

以下のような問題を検知できる

- テンプレートインジェクション
- 認証情報の永続化設定
- 適切な権限スコープの設定

actionlint や ghalint と組み合わせて CI や pre-commit hook で利用するのが良さそう


## Datadog Bits AI SRE

Datadog の Bits AI が Investigation したモニターで発生したアラートの原因を分析してくれる

分析できるモニタータイプには制限がある

モニター起因で Runbook の実行につなげることもできる


## GitHub Merge Queue

GitHub ではマージ前に Merge Queue という機能を利用することができる

利用することでマージ前にマージ先のブランチでの CI がパスすることを確認したうえでマージすることができる

複数の PR がある場合はキューイングされ順番に実行される

有効化するにはワークフローでの設定修正が必要

また PR でのパイプライン -> マージキューでのパイプライン -> マージでのパイプラインと実行される

その分時間がかかることになるので注意は必要

失敗した場合のフィードバックはどうなるんだろうか
